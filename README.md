# Iceland - Golden Circle

This repository contains a simple starter site and assets for the "Golden Circle" tourist route in Iceland. It includes a brief overview, a landing page, and an open-source MIT license. Use this repo as the basis for a small static website, tour information, or image gallery showcasing the Golden Circle.

Contents:
- index.html — simple landing page
- LICENSE — MIT license

About
A tiny starter repo for showcasing the Golden Circle (Þingvellir, Geysir, Gullfoss) with images and short descriptions.
